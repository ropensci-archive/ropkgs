ropkgs
=======

`ropkgs` is an R client to discover and install rOpensci packages.

## Installation


```r
install.packages("devtools")
devtools::install_github("sckott/ropkgs")
```


```r
library("ropkgs")
```

## List all packages


```r
adfadf
```

## Search for a package


```r
adfadf
```

## Install packages


```r
afafasf
```

## Uninstall packages


```r
uninstall
```
